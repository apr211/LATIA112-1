#Part1.
Azure 提供的主要服務：
文本分析：這項服務提供了情感分析、關鍵詞提取、實體識別（比如人名、地點、組織名稱）等功能。
翻譯：支持多語言翻譯，可應用於實時或批量文本翻譯。
語言理解 (LUIS)：允許開發者創建應用，這些應用能理解用戶的自然語言輸入，並做出相應的反應。
語音服務：包括語音轉文本、文本轉語音以及語音翻譯。
QnA Maker：能夠從半結構化的內容（如常見問題集、產品手冊等）中創建問答式的聊天機器人。
